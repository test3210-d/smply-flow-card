### Можливості

- Два сонячні входи
- Дві батареї
- До трьох мереж
- Два споживачі
- Нативний редактор
- Автоматичний Solar Total
- Активні анімовані потоки

### Напрямок батареї за замовченням

- Заряд: Центр → Батарея
- Розряд: Батарея → Центр
